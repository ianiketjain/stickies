# Stickies

# Steps to run on local

1. npm i
2. npm run start
3. for running on local system must have Node v18


# Steps to run on local docker 
1. docker build -t stickies .
2. docker run -p 4000:3000 stickies
3. go to URL http://localhost:4000/
